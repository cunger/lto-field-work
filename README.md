## Run locally

```
$ npx expo start
```

## Build

Tailwind:
```
$ npm run build:tailwind
```

Translations:
```
$ npm run build:translations
```

App:
```
$ npm run bundle:ios
$ npm run eject
```

## Deploy

Building and submitting the app:
```
$ eas build --platform ios --profile production 
$ eas submit --platform ios
```
