## Build pipeline

Tailwind:
```
$ npm run build:tailwind
```

Translations:
```
$ npm run build:translations
```

## Test

```
$ npx expo start
```

## Deploy

Ejecting:
```
$ npm run eject
```

Building and submitting the app:
```
$ eas build --profile production --platform ios
$ eas submit --profile production --platform ios
```
